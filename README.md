# Fullstack Starter

## What's in it?

- [x] NextJS
- [x] GraphQL
- [x] TypeScript
- [x] Prisma
- [x] Chakra-UI
- [x] Cypress
- [x] Jest
- [x] Husky

## How to run

1. Install the dependencies.

```bash
yarn install
```

2. Run the app

```bash
yarn dev
```

### Build

```bash
yarn build
```

### Build with Bundle Analyzer

```bash
yarn build-analyze
```
